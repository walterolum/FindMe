# FindMe — Cavendish University Uganda Lost & Found Management System

A modern, intelligent platform for reporting, tracking, and reclaiming lost belongings across all Cavendish University Uganda campuses.

## Features

- **Single HTML file** — No server required, runs entirely in the browser
- **6 User Roles** — Student, Staff, Security, Moderator, Admin, Super Admin
- **13 Database Tables** — Full SQLite database via sql.js (WebAssembly)
- **45+ Page Methods** — Complete SPA with client-side routing
- **Real CU Data** — 5 faculties, 20+ courses, 3 campuses
- **Smart Item Matching** — Auto-match lost & found by AI scoring
- **Admin Audit Log** — Track all system actions
- **Announcements** — Target by role, priority, campus
- **Analytics Dashboard** — Category, faculty, location breakdowns
- **Data Export** — CSV/JSON for all tables
- **Dark Glassmorphism UI** — Modern, responsive design
- **QR Code Generation** — On every item detail page

## Tech Stack

- HTML5
- CSS3 (Glassmorphism, animations, responsive grid)
- JavaScript ES6+ (SPA routing, client-side rendering)
- sql.js (SQLite compiled to WebAssembly)
- Bootstrap Icons

## Getting Started

1. Clone this repository
2. Open `frontend/index.html` in your browser
3. Or access via: `http://localhost/lost-and-found/frontend/index.html`
